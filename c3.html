Administrer des systèmes

Site dynamique avec base de données

Cahier des charges

Ce projet universitaire visait à créer un site Web comportant les fonctionnalités suivantes :

  • Pages customisées avec du CSS
  • Espace administrateur avec authentification
  • Authentification d'un utilisateur
  • Page d'inscription d'un utilisateur
  • Suppression des comptes non finalisés

Nous avions le choix du thème du site et des technologies utilisées.

Développement

J'ai hébergé ce site sur un serveur Apache. En front-end, j'ai utilisé le framework Bootstrap ainsi que les langages HTML, CSS et JavaScript. Pour concevoir le système d'authentification, j'ai dû utiliser MySQL et PHP pour l'envoi de formulaires.

Avantages

Ce projet m'a appris les bases du développement back-end et à structurer un site. Je suis capable de créer un formulaire, un système de connexion et un lien avec une base de données.
